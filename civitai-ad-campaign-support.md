# Civitai Ads Campaign Support

We want to add support to create civitai ads campaign so users can have ready to upload images to our advertising platform and they show properly in the civitai website. Here are the supported dimensions:

```
[
  {
    "name": "Footer",
    "sizes": {
      "mobile": [[320, 50]],
      "desktop": [[728, 90], [970, 90]]
    }
  },
  {
    "name": "Banner",
    "sizes": {
      "mobile": [[300, 250]],
      "desktop": [[728, 90], [970, 250]]
    }
  },
  {
    "name": "Rectangle",
    "sizes": {
      "any": [[300, 250]]
    }
  },
  {
    "name": "Skyscraper",
    "sizes": {
      "any": [[300, 600]]
    }
  }
]
```
